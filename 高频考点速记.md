# 初级会计职称考试 - 高频考点速记

## 📚 会计实务高频考点

### 一、会计基础（必考 10-15 分）

#### 1. 会计要素
```
资产 = 负债 + 所有者权益（静态等式）
收入 - 费用 = 利润（动态等式）
```

#### 2. 会计科目分类
| 类别 | 借方 | 贷方 | 余额方向 |
|------|------|------|----------|
| 资产类 | 增加 | 减少 | 借方 |
| 负债类 | 减少 | 增加 | 贷方 |
| 所有者权益 | 减少 | 增加 | 贷方 |
| 收入类 | 减少 | 增加 | 无余额 |
| 费用类 | 增加 | 减少 | 无余额 |

#### 3. 借贷记账法规则
- **有借必有贷，借贷必相等**
- 资产 + 费用 = 负债 + 所有者权益 + 收入

---

### 二、资产（必考 30-35 分）⭐⭐⭐

#### 1. 货币资金
```
库存现金 + 银行存款 + 其他货币资金
```

#### 2. 应收账款入账价值
```
应收账款 = 销售价款 + 增值税 + 代垫费用 - 商业折扣
```
⚠️ 现金折扣不影响应收账款入账价值

#### 3. 存货成本
```
存货采购成本 = 购买价款 + 相关税费 + 运输费 + 装卸费 + 保险费
              + 运输途中合理损耗 + 入库前挑选整理费
```
⚠️ 非正常损耗不计入成本

#### 4. 固定资产折旧方法
| 方法 | 公式 |
|------|------|
| 年限平均法 | (原价 - 预计净残值) ÷ 预计使用年限 |
| 工作量法 | 单位工作量折旧额 × 实际工作量 |
| 双倍余额递减法 | 期初账面净值 × (2÷预计使用年限) |
| 年数总和法 | (原价 - 净残值) × 尚可使用年限÷年数总和 |

#### 5. 固定资产处置损益
```
处置损益 = 处置收入 - (原价 - 累计折旧 - 减值准备) - 清理费用
```

#### 6. 无形资产
- 使用寿命有限的无形资产：**需要摊销**
- 使用寿命不确定的无形资产：**不摊销，每年减值测试**

---

### 三、负债（必考 8-10 分）

#### 1. 应付职工薪酬核算内容
```
工资 + 奖金 + 津贴 + 福利费 + 社会保险 + 住房公积金
+ 工会经费 + 职工教育经费 + 非货币性福利 + 辞退福利
```

#### 2. 应交增值税
```
应纳税额 = 销项税额 - 进项税额
销项税额 = 销售额 × 税率
```

---

### 四、所有者权益（必考 6-8 分）

#### 1. 留存收益
```
留存收益 = 盈余公积 + 未分配利润
```

#### 2. 盈余公积提取
```
法定盈余公积：净利润的 10%
任意盈余公积：股东会决定
```
⚠️ 法定盈余公积累计达到注册资本 50% 可不再提取

#### 3. 利润分配顺序
```
1. 弥补以前年度亏损
2. 提取法定盈余公积
3. 提取任意盈余公积
4. 向投资者分配利润
```

---

### 五、收入费用利润（必考 12-15 分）⭐⭐⭐

#### 1. 收入确认条件（5 个同时满足）
1. 合同各方已批准并承诺履行义务
2. 合同明确了权利和义务
3. 合同有明确的支付条款
4. 合同具有商业实质
5. 对价很可能收回

#### 2. 利润计算
```
营业利润 = 营业收入 - 营业成本 - 税金及附加 - 期间费用
          - 资产减值损失 + 公允价值变动收益 + 投资收益

利润总额 = 营业利润 + 营业外收入 - 营业外支出

净利润 = 利润总额 - 所得税费用
```

---

### 六、财务报表（必考 8-10 分）

#### 1. 资产负债表填列方法
| 项目 | 填列方法 |
|------|----------|
| 货币资金 | 库存现金 + 银行存款 + 其他货币资金 |
| 应收账款 | 应收账款明细账借方余额 + 预收账款明细账借方余额 - 坏账准备 |
| 预付款项 | 预付账款明细账借方余额 + 应付账款明细账借方余额 - 坏账准备 |
| 存货 | 原材料 + 库存商品 + 生产成本 + 周转材料 - 存货跌价准备 |
| 固定资产 | 固定资产 - 累计折旧 - 固定资产减值准备 |

#### 2. 利润表项目
- **营业收入** = 主营业务收入 + 其他业务收入
- **营业成本** = 主营业务成本 + 其他业务成本

## ⚖️ 经济法高频考点

### 一、会计法律制度（必考 8-10 分）

#### 1. 会计档案保管期限
| 类型 | 保管期限 |
|------|----------|
| 会计凭证 | 30 年 |
| 会计账簿 | 30 年 |
| 财务会计报告 | 永久（年度）/ 10 年（月度、季度） |
| 其他 | 10 年 / 30 年 |

#### 2. 会计人员回避制度
- 单位负责人的**直系亲属**不得担任会计机构负责人
- 会计机构负责人的**直系亲属**不得担任出纳

---

### 二、支付结算（必考 12-15 分）⭐⭐⭐

#### 1. 票据权利时效
| 票据类型 | 时效 |
|----------|------|
| 商业汇票 | 到期日起 2 年 |
| 银行汇票 | 出票日起 2 年 |
| 支票 | 出票日起 6 个月 |
| 追索权 | 被拒绝承兑/付款日起 6 个月 |

#### 2. 票据提示付款期限
| 票据类型 | 期限 |
|----------|------|
| 银行汇票 | 出票日起 1 个月 |
| 商业汇票 | 到期日起 10 天 |
| 银行本票 | 出票日起 2 个月 |
| 支票 | 出票日起 10 天 |

#### 3. 银行账户
| 账户类型 | 用途 |
|----------|------|
| 基本存款账户 | 主办账户，可存取现金 |
| 一般存款账户 | 借款转存/归还，可存现不可取现 |
| 专用存款账户 | 专项资金管理 |
| 临时存款账户 | 临时需要，有效期≤2 年 |

---

### 三、增值税（必考 12-15 分）⭐⭐⭐

#### 1. 增值税税率
| 税率 | 适用范围 |
|------|----------|
| 13% | 销售货物、加工修理修配劳务、有形动产租赁 |
| 9% | 交通运输、邮政、基础电信、建筑、不动产、土地使用权 |
| 6% | 现代服务（除租赁）、生活服务、金融服务、增值电信 |
| 0% | 出口货物、服务 |

#### 2. 征收率
- 小规模纳税人：**3%**
- 个人出租住房：**1.5%**

#### 3. 进项税额不得抵扣
- 用于简易计税方法计税项目
- 用于免征增值税项目
- 用于集体福利或个人消费
- 非正常损失的购进货物

---

### 四、所得税（必考 12-15 分）⭐⭐⭐

#### 1. 企业所得税税率
| 税率 | 适用对象 |
|------|----------|
| 25% | 一般企业（基本税率） |
| 20% | 小型微利企业 |
| 15% | 高新技术企业 |

#### 2. 扣除标准
| 项目 | 扣除标准 |
|------|----------|
| 职工福利费 | 工资薪金总额×14% |
| 工会经费 | 工资薪金总额×2% |
| 职工教育经费 | 工资薪金总额×8% |
| 业务招待费 | 发生额×60% 且≤营业收入×5‰ |
| 广告费和业务宣传费 | 营业收入×15% |
| 公益性捐赠 | 利润总额×12% |

#### 3. 个人所得税综合所得
```
综合所得 = 工资薪金 + 劳务报酬 + 稿酬 + 特许权使用费
```

#### 4. 专项附加扣除（6 项）
1. 子女教育：1000 元/月/孩
2. 继续教育：400 元/月或 3600 元/年
3. 大病医疗：限额 80000 元/年
4. 住房贷款利息：1000 元/月
5. 住房租金：800-1500 元/月
6. 赡养老人：2000 元/月

---

### 五、劳动合同（必考 8-10 分）⭐⭐

#### 1. 试用期规定
| 合同期限 | 试用期上限 |
|----------|------------|
| <3 个月 | 不得约定 |
| 3 个月 -1 年 | ≤1 个月 |
| 1 年 -3 年 | ≤2 个月 |
| ≥3 年或无固定 | ≤6 个月 |

⚠️ 试用期工资≥同岗位最低档工资或约定工资的**80%**，且≥最低工资标准

#### 2. 经济补偿金
```
经济补偿 = 工作年限 × 月工资
```
- 工作满 1 年支付 1 个月工资
- 6 个月以上不满 1 年按 1 年计算
- 不满 6 个月支付半个月工资

## 💡 记忆口诀

### 会计要素
> 资负所，收费用，六大要素要记牢
> 资产负债所有者，财务状况静态表
> 收入费用和利润，经营成果动态报

### 借贷记账法
> 资产费用借增贷，负债权益收借减
> 有借必有贷必等，记账规则记心间

### 存货成本
> 买价税费运装保，合理损耗挑选费
> 非正常损不计入，入库之前都算数

### 增值税税率
> 一三货物加修理，有形动产也十三
> 零九交通邮建筑，不动土地九来算
> 六服金生现电信，生活服务六最便
