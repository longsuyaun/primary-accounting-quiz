# 初级会计职称考试 - 易错点汇总

## 📚 会计实务易错点

### 一、资产类易错点

#### ❌ 错误 1：商业折扣 vs 现金折扣
| 项目 | 商业折扣 | 现金折扣 |
|------|----------|----------|
| **定义** | 促销打折 | 提前付款优惠 |
| **会计处理** | 按扣除后金额入账 | 不影响应收账款入账价值 |
| **分录** | 借：应收账款（折扣后）<br>贷：主营业务收入 | 借：银行存款<br>财务费用（折扣额）<br>贷：应收账款 |

#### ❌ 错误 2：存货非正常损耗
- **错误做法**：计入存货成本
- **正确做法**：计入当期损益（营业外支出或管理费用）

#### ❌ 错误 3：固定资产折旧范围
| 情况 | 是否计提折旧 |
|------|--------------|
| 季节性停用 | ✅ 计提 |
| 大修理停用 | ✅ 计提 |
| 未使用房屋 | ✅ 计提 |
| 已提足折旧继续使用 | ❌ 不计提 |
| 经营租入设备 | ❌ 不计提 |
| 融资租出设备 | ❌ 不计提 |
| 单独计价入账土地 | ❌ 不计提 |

#### ❌ 错误 4：无形资产摊销科目
- **错误做法**：一律计入管理费用
- **正确做法**：
  - 管理用 → 管理费用
  - 生产用 → 制造费用
  - 出租用 → 其他业务成本

---

### 二、负债类易错点

#### ❌ 错误 5：应付职工薪酬核算范围
**包括**：
- ✅ 工资、奖金、津贴
- ✅ 职工福利费
- ✅ 五险一金
- ✅ 工会经费、职工教育经费
- ✅ 非货币性福利
- ✅ 辞退福利

**不包括**：
- ❌ 职工出差报销的差旅费
- ❌ 职工个人负担的社保

#### ❌ 错误 6：增值税视同销售
**视同销售的情形**：
- ✅ 将自产、委托加工货物用于集体福利
- ✅ 将自产、委托加工货物用于个人消费
- ✅ 将自产、委托加工货物用于投资
- ✅ 将自产、委托加工货物用于分配
- ✅ 将自产、委托加工货物用于无偿赠送

**不视同销售**：
- ❌ 将外购货物用于集体福利
- ❌ 将外购货物用于个人消费

### 三、所有者权益易错点

#### ❌ 错误 7：留存收益变动
| 业务 | 留存收益总额 |
|------|--------------|
| 提取盈余公积 | 不变（内部结转） |
| 盈余公积转增资本 | ⬇️ 减少 |
| 盈余公积补亏 | 不变（内部结转） |
| 宣告发放现金股利 | ⬇️ 减少 |
| 宣告发放股票股利 | 不变 |

#### ❌ 错误 8：资本公积 vs 留存收益
- **资本公积**：资本溢价、股本溢价、其他资本公积
- **留存收益**：盈余公积、未分配利润
- **易错点**：资本公积不属于留存收益

---

### 四、收入费用易错点

#### ❌ 错误 9：收入确认时点
| 销售方式 | 确认时点 |
|----------|----------|
| 直接收款 | 收到货款或取得收款凭证 |
| 托收承付 | 办妥托收手续 |
| 预收款 | 发出商品 |
| 委托代销（手续费） | 收到代销清单 |
| 分期收款 | 合同约定收款日 |

#### ❌ 错误 10：营业外收入 vs 其他业务收入
| 项目 | 正确科目 |
|------|----------|
| 销售材料收入 | 其他业务收入 |
| 出租固定资产租金 | 其他业务收入 |
| 出租包装物租金 | 其他业务收入 |
| 出售固定资产净收益 | 资产处置损益 |
| 出售无形资产净收益 | 资产处置损益 |
| 盘盈利得 | 营业外收入 |
| 捐赠利得 | 营业外收入 |
| 罚款收入 | 营业外收入 |

### 五、财务报表易错点

#### ❌ 错误 11：资产负债表项目填列
| 项目 | 易错点 |
|------|--------|
| 应收账款 | =应收账款借方 + 预收账款借方 - 坏账准备 |
| 预付款项 | =预付账款借方 + 应付账款借方 - 坏账准备 |
| 应付账款 | =应付账款贷方 + 预付账款贷方 |
| 预收款项 | =预收账款贷方 + 应收账款贷方 |
| 存货 | 需要减去存货跌价准备 |
| 固定资产 | = 固定资产 - 累计折旧 - 减值准备 |

#### ❌ 错误 12：利润表计算
```
营业利润 ≠ 利润总额
营业利润 = 营业收入 - 营业成本 - 税金及附加 - 期间费用 - 资产减值损失 + 公允价值变动收益 + 投资收益
利润总额 = 营业利润 + 营业外收入 - 营业外支出
净利润 = 利润总额 - 所得税费用
```
⚠️ 营业外收支不影响营业利润！

## ⚖️ 经济法易错点

### 一、支付结算易错点

#### ❌ 错误 13：票据基本当事人
| 票据类型 | 基本当事人 |
|----------|------------|
| 商业汇票 | 出票人、付款人、收款人 |
| 银行汇票 | 出票人、付款人、收款人 |
| 银行本票 | 出票人、收款人 |
| 支票 | 出票人、付款人、收款人 |

⚠️ 背书人、承兑人、保证人是**非基本当事人**

#### ❌ 错误 14：票据权利时效起算点
| 权利类型 | 起算点 |
|----------|--------|
| 付款请求权 | 票据到期日 |
| 追索权 | 被拒绝承兑/付款日 |
| 再追索权 | 清偿日或被起诉日 |

#### ❌ 错误 15：银行账户现金支取
| 账户类型 | 可否取现 |
|----------|----------|
| 基本存款账户 | ✅ 可以 |
| 一般存款账户 | ❌ 不可以 |
| 专用存款账户 | 视情况而定 |
| 临时存款账户 | ✅ 可以 |

---

### 二、增值税易错点

#### ❌ 错误 16：混合销售 vs 兼营
| 概念 | 定义 | 税务处理 |
|------|------|----------|
| 混合销售 | 一项销售行为既涉及货物又涉及服务 | 按主业适用税率 |
| 兼营 | 经营多种税率项目 | 分别核算，从高适用 |

**举例**：
- 混合销售：销售设备并安装（一项行为）
- 兼营：商场销售商品 + 餐饮服务（多项行为）

#### ❌ 错误 17：进项税额抵扣
**不得抵扣的情形**：
- ❌ 用于简易计税方法计税项目
- ❌ 用于免征增值税项目
- ❌ 用于集体福利
- ❌ 用于个人消费
- ❌ 非正常损失的购进货物
- ❌ 非正常损失的在产品、产成品

---

### 三、所得税易错点

#### ❌ 错误 18：免税收入 vs 不征税收入
| 类型 | 项目 |
|------|------|
| **免税收入** | 国债利息收入、符合条件的股息红利 |
| **不征税收入** | 财政拨款、行政事业性收费、政府性基金 |

#### ❌ 错误 19：扣除限额计算基数
| 项目 | 计算基数 |
|------|----------|
| 业务招待费 | 营业收入 |
| 广告费和业务宣传费 | 营业收入 |
| 公益性捐赠 | 利润总额 |
| 职工福利费 | 工资薪金总额 |
| 工会经费 | 工资薪金总额 |
| 职工教育经费 | 工资薪金总额 |

---

### 四、劳动合同易错点

#### ❌ 错误 20：试用期规定
| 易错点 | 正确规定 |
|--------|----------|
| 试用期包含在合同期内吗？ | ✅ 包含 |
| 单独约定试用期合同？ | ❌ 不成立，视为劳动合同期限 |
| 试用期工资标准？ | ≥同岗位 80% 且≥最低工资 |
| 试用期可以约定几次？ | 只能约定一次 |
| 试用期可以解除劳动合同吗？ | 可以，但需证明不符合录用条件 |

#### ❌ 错误 21：经济补偿金 vs 赔偿金
| 项目 | 经济补偿金 | 赔偿金 |
|------|------------|--------|
| **适用情形** | 合法解除 | 违法解除 |
| **计算标准** | N×月工资 | 2N×月工资 |
| **N 的含义** | 工作年限 | 工作年限 |

## 💡 记忆技巧

- 商业折扣 vs 现金折扣

重要数字单独记忆：
- 支票提示付款期：**10 天**

- 试用期工资比例：**80%**
- 法定盈余公积提取比例：**10%**
