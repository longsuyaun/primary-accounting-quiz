# 初级会计职称考试 - 计算公式大全

## 📚 会计实务计算公式

### 一、资产类计算

#### 1. 应收账款入账价值
```
应收账款 = 销售价款 + 增值税销项税额 + 代垫运杂费 - 商业折扣
```
⚠️ 现金折扣不影响入账价值

#### 2. 存货采购成本
```
存货采购成本 = 购买价款 + 相关税费 + 运输费 + 装卸费 + 保险费
              + 运输途中合理损耗 + 入库前挑选整理费 - 商业折扣
```

#### 3. 材料成本差异率
```
材料成本差异率 = (期初差异 + 本期差异) ÷ (期初计划成本 + 本期入库计划成本) × 100%

发出材料应负担差异 = 发出材料计划成本 × 材料成本差异率

发出材料实际成本 = 发出材料计划成本 ± 发出材料应负担差异
```

#### 4. 固定资产折旧

**年限平均法（直线法）**：
```
年折旧额 = (原价 - 预计净残值) ÷ 预计使用年限
月折旧额 = 年折旧额 ÷ 12
年折旧率 = (1 - 预计净残值率) ÷ 预计使用年限 × 100%
```

**工作量法**：
```
单位工作量折旧额 = (原价 - 预计净残值) ÷ 预计总工作量
某项固定资产月折旧额 = 该项固定资产当月工作量 × 单位工作量折旧额
```

**双倍余额递减法**：
```
年折旧率 = 2 ÷ 预计使用年限 × 100%
年折旧额 = 期初固定资产账面净值 × 年折旧率
月折旧额 = 年折旧额 ÷ 12
```
⚠️ 最后两年改为年限平均法

**年数总和法**：
```
年折旧率 = 尚可使用年限 ÷ 预计使用年限的年数总和 × 100%
年折旧额 = (原价 - 预计净残值) × 年折旧率
月折旧额 = 年折旧额 ÷ 12
```

#### 5. 固定资产处置损益
```
处置损益 = 处置收入 - (固定资产原价 - 累计折旧 - 减值准备) - 清理费用
```
- 正数：处置收益 → 资产处置损益（贷方）
- 负数：处置损失 → 资产处置损益（借方）

#### 6. 无形资产摊销
```
年摊销额 = (成本 - 预计净残值 - 减值准备) ÷ 预计使用年限
月摊销额 = 年摊销额 ÷ 12
```

### 二、负债类计算

#### 7. 应付职工薪酬
```
应付职工薪酬 = 工资 + 奖金 + 津贴 + 福利费 + 社会保险 + 住房公积金
              + 工会经费 + 职工教育经费 + 非货币性福利 + 辞退福利
```

#### 8. 增值税计算

**一般纳税人**：
```
应纳税额 = 当期销项税额 - 当期进项税额
销项税额 = 销售额 × 税率
销售额 = 含税销售额 ÷ (1 + 税率)
```

**小规模纳税人**：
```
应纳税额 = 销售额 × 征收率 (3%)
销售额 = 含税销售额 ÷ (1 + 征收率)
```

---

### 三、所有者权益计算

#### 9. 盈余公积提取
```
法定盈余公积 = 净利润 × 10%
任意盈余公积 = 净利润 × 股东会决定比例
```
⚠️ 法定盈余公积累计达到注册资本 50% 可不再提取

#### 10. 未分配利润
```
期末未分配利润 = 期初未分配利润 + 本期净利润 - 提取盈余公积 - 分配利润
```

---

### 四、收入费用利润计算

#### 11. 利润计算
```
营业利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用
          - 资产减值损失 + 公允价值变动收益（- 损失）+ 投资收益（- 损失）

利润总额 = 营业利润 + 营业外收入 - 营业外支出

净利润 = 利润总额 - 所得税费用
```

#### 12. 完工百分比法
```
完工进度 = 已发生成本 ÷ 预计总成本 × 100%
本期确认收入 = 合同总收入 × 完工进度 - 以前期间累计已确认收入
本期确认成本 = 合同预计总成本 × 完工进度 - 以前期间累计已确认成本
```

---

### 五、管理会计计算

#### 13. 本量利分析
```
单位贡献毛益 = 单价 - 单位变动成本
贡献毛益总额 = 销售收入 - 变动成本总额

保本点销售量 = 固定成本 ÷ 单位贡献毛益
保本点销售额 = 固定成本 ÷ 贡献毛益率

贡献毛益率 = 单位贡献毛益 ÷ 单价 × 100%
```

#### 14. 目标利润
```
目标利润销售量 = (固定成本 + 目标利润) ÷ 单位贡献毛益
目标利润销售额 = (固定成本 + 目标利润) ÷ 贡献毛益率
```

## ⚖️ 经济法计算公式

### 一、增值税计算

#### 15. 一般纳税人增值税
```
应纳税额 = 销项税额 - 进项税额
销项税额 = 销售额 × 税率
```

#### 16. 进口货物增值税
```
组成计税价格 = 关税完税价格 + 关税 + 消费税
应纳税额 = 组成计税价格 × 增值税税率
```

---

### 二、消费税计算

#### 17. 从价定率
```
应纳税额 = 销售额 × 比例税率
```

#### 18. 从量定额
```
应纳税额 = 销售数量 × 定额税率
```

#### 19. 复合计税
```
应纳税额 = 销售额 × 比例税率 + 销售数量 × 定额税率
```

#### 20. 组成计税价格
```
自产自用：组成计税价格 = (成本 + 利润) ÷ (1 - 消费税税率)
委托加工：组成计税价格 = (材料成本 + 加工费) ÷ (1 - 消费税税率)
进口：组成计税价格 = (关税完税价格 + 关税) ÷ (1 - 消费税税率)
```

---

### 三、所得税计算

#### 21. 企业所得税
```
应纳税额 = 应纳税所得额 × 税率
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 以前年度亏损
```

#### 22. 个人所得税（综合所得）
```
应纳税所得额 = 年收入 - 60000 元 - 专项扣除 - 专项附加扣除 - 其他扣除

应纳税额 = 应纳税所得额 × 适用税率 - 速算扣除数
```

#### 23. 专项附加扣除标准
| 项目 | 扣除标准 |
|------|----------|
| 子女教育 | 1000 元/月/孩 |
| 继续教育 | 400 元/月或 3600 元/年 |
| 大病医疗 | 限额 80000 元/年 |
| 住房贷款利息 | 1000 元/月 |
| 住房租金 | 800-1500 元/月 |
| 赡养老人 | 2000 元/月 |

---

### 四、劳动合同计算

#### 24. 经济补偿金
```
经济补偿 = 工作年限 × 月工资

工作年限计算：
- 满 1 年支付 1 个月
- 6 个月以上不满 1 年按 1 年
- 不满 6 个月支付半个月
```

#### 25. 赔偿金
```
赔偿金 = 2 × 经济补偿金
```
⚠️ 适用于违法解除劳动合同

## 💡 公式记忆技巧

### 1. 分类记忆
将公式按类别分组：
- 资产类公式
- 负债类公式
- 利润类公式
- 税法类公式

### 2. 理解记忆
理解公式背后的逻辑：
- 折旧公式：将资产价值分摊到各期
- 利润公式：收入 - 费用 = 利润
- 增值税公式：销项 - 进项 = 应纳税

### 3. 对比记忆
相似公式对比：
- 直线法 vs 加速折旧法
- 从价计征 vs 从量计征
- 经济补偿 vs 赔偿金
