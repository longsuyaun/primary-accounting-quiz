# 初级会计职称考试 - 会计分录汇总

## 📚 资产类分录

### 一、货币资金

#### 1. 现金存取
```
存现：
借：银行存款
  贷：库存现金

提现：
借：库存现金
  贷：银行存款
```

#### 2. 其他货币资金
```
存入投资款：
借：其他货币资金——存出投资款
  贷：银行存款

购买股票：
借：交易性金融资产
  贷：其他货币资金——存出投资款
```

---

### 二、应收及预付款项

#### 3. 应收账款
```
销售商品：
借：应收账款
  贷：主营业务收入
      应交税费——应交增值税（销项税额）

收回货款：
借：银行存款
  贷：应收账款
```

#### 4. 坏账准备
```
计提坏账：
借：信用减值损失
  贷：坏账准备

发生坏账：
借：坏账准备
  贷：应收账款

坏账收回：
借：应收账款
  贷：坏账准备
借：银行存款
  贷：应收账款
```

---

### 三、存货

#### 5. 原材料购入
```
购入材料：
借：材料采购（实际成本）
  贷：银行存款

入库：
借：原材料（计划成本）
    材料成本差异（超支）
  贷：材料采购
      材料成本差异（节约）
```

#### 6. 发出材料
```
借：生产成本
    制造费用
    管理费用
  贷：原材料

结转差异：
借：生产成本等
  贷：材料成本差异（节约用红字）
```

#### 7. 库存商品
```
完工入库：
借：库存商品
  贷：生产成本

销售结转成本：
借：主营业务成本
  贷：库存商品
```

#### 8. 存货跌价准备
```
计提跌价：
借：资产减值损失
  贷：存货跌价准备

转回跌价：
借：存货跌价准备
  贷：资产减值损失
```

---

### 四、固定资产

#### 9. 固定资产取得
```
外购不需要安装：
借：固定资产
  贷：银行存款

外购需要安装：
借：在建工程
  贷：银行存款

安装完成：
借：固定资产
  贷：在建工程
```

#### 10. 固定资产折旧
```
计提折旧：
借：制造费用（生产用）
    管理费用（管理用）
    销售费用（销售用）
  贷：累计折旧
```

#### 11. 固定资产处置
```
转入清理：
借：固定资产清理
    累计折旧
    固定资产减值准备
  贷：固定资产

发生清理费：
借：固定资产清理
  贷：银行存款

收到价款：
借：银行存款
  贷：固定资产清理
      应交税费——应交增值税（销项税额）

结转损益：
借：固定资产清理
  贷：资产处置损益（收益）
或
借：资产处置损益（损失）
  贷：固定资产清理
```

---

### 五、无形资产

#### 12. 无形资产取得
```
外购：
借：无形资产
  贷：银行存款
```

#### 13. 无形资产摊销
```
摊销：
借：管理费用（管理用）
    制造费用（生产用）
    其他业务成本（出租用）
  贷：累计摊销
```

---

## 📚 负债类分录

### 六、应付职工薪酬

#### 14. 计提工资
```
计提：
借：生产成本
    制造费用
    管理费用
    销售费用
  贷：应付职工薪酬——工资

发放：
借：应付职工薪酬——工资
  贷：银行存款
      应交税费——应交个人所得税
      其他应付款——社保（个人部分）
```

#### 15. 社保及公积金
```
计提单位部分：
借：生产成本等
  贷：应付职工薪酬——社会保险
      应付职工薪酬——住房公积金

缴纳：
借：应付职工薪酬——社会保险（单位）
    其他应付款——社保（个人）
  贷：银行存款
```

---

### 七、应交税费

#### 16. 增值税
```
采购：
借：原材料
    应交税费——应交增值税（进项税额）
  贷：银行存款

销售：
借：银行存款
  贷：主营业务收入
      应交税费——应交增值税（销项税额）

缴纳：
借：应交税费——应交增值税（已交税金）
  贷：银行存款
```

#### 17. 消费税
```
计提：
借：税金及附加
  贷：应交税费——应交消费税

缴纳：
借：应交税费——应交消费税
  贷：银行存款
```

---

## 📚 所有者权益分录

### 八、实收资本

#### 18. 接受投资
```
接受货币投资：
借：银行存款
  贷：实收资本

接受非货币投资：
借：固定资产等
  贷：实收资本
      资本公积——资本溢价
```

---

### 九、留存收益

#### 19. 提取盈余公积
```
提取法定盈余公积：
借：利润分配——提取法定盈余公积
  贷：盈余公积——法定盈余公积

提取任意盈余公积：
借：利润分配——提取任意盈余公积
  贷：盈余公积——任意盈余公积
```

#### 20. 盈余公积使用
```
转增资本：
借：盈余公积
  贷：实收资本

弥补亏损：
借：盈余公积
  贷：利润分配——盈余公积补亏
```

#### 21. 利润分配
```
宣告分配现金股利：
借：利润分配——应付现金股利
  贷：应付股利

实际支付：
借：应付股利
  贷：银行存款

结转未分配利润：
借：利润分配——未分配利润
  贷：利润分配——提取法定盈余公积
      利润分配——应付现金股利
```

## 📚 收入费用分录

### 十、收入

#### 22. 销售商品
```
确认收入：
借：银行存款/应收账款
  贷：主营业务收入
      应交税费——应交增值税（销项税额）

结转成本：
借：主营业务成本
  贷：库存商品
```

#### 23. 其他业务收入
```
销售材料：
借：银行存款
  贷：其他业务收入
      应交税费——应交增值税（销项税额）

结转成本：
借：其他业务成本
  贷：原材料
```

---

### 十一、费用

#### 24. 期间费用
```
管理费用：
借：管理费用
  贷：银行存款/应付职工薪酬等

销售费用：
借：销售费用
  贷：银行存款等

财务费用：
借：财务费用
  贷：银行存款/应付利息等
```

---

## 📚 利润分录

### 十二、利润形成

#### 25. 结转损益
```
结转收入：
借：主营业务收入
    其他业务收入
    营业外收入
  贷：本年利润

结转费用：
借：本年利润
  贷：主营业务成本
      税金及附加
      管理费用
      销售费用
      财务费用
      营业外支出
      所得税费用
```

#### 26. 所得税
```
计提所得税：
借：所得税费用
  贷：应交税费——应交所得税

结转：
借：本年利润
  贷：所得税费用
```

#### 27. 结转净利润
```
盈利：
借：本年利润
  贷：利润分配——未分配利润

亏损：
借：利润分配——未分配利润
  贷：本年利润
```

## 💡 记忆技巧

### 1. 借贷规律
- **资产、费用**：借增贷减
- **负债、权益、收入**：借减贷增

### 2. 对应关系
掌握常见对应科目：
- 银行存款 ↔ 收入/费用/资产
- 应收账款 ↔ 主营业务收入
- 应付职工薪酬 ↔ 成本费用

### 3. 业务循环
按业务流程记忆：
- 采购 → 入库 → 生产 → 销售 → 收款
