## 📅 备考计划

### 第一阶段：基础学习（2 周）
- 通读教材，理解基本概念
- 完成章节练习（每天 30 题）
- 标记不懂的知识点

### 第二阶段：强化提高（2 周）
- 重点突破高分值章节

### 第三阶段：冲刺模拟（1 周）
- 模拟考试（每天 1 套）
- 复习错题本
- 背诵公式和分录

## 💡 学习建议

### 1. 会计实务
- **重点**：资产、收入费用、财务报表
- **方法**：理解分录逻辑，多做练习
- **技巧**：掌握 T 型账户，理解借贷关系

### 2. 经济法
- **重点**：支付结算、增值税、所得税
- **方法**：理解法条背后的逻辑
- **技巧**：数字记忆、对比记忆

### 3. 通用技巧
- 每天至少练习 50 道题
- 建立错题本，定期复习
- 考前一周重点看易错点
